发表于 2018-10-08 | 分类于 cs231n | 次

优化算法总结

简介

主要包括:

SGD, Momentum, Nesterov Momentum, AdaGrad, RMSProp, Adam

原 note 地址：

http://cs231n.github.io/neural-networks-3/#sgd

BGD batch gradient descent

参数：学习率 $\varepsilon$

计算：将数据集的每一个样本带入梯度公式，求平均后更新参数

$w = w-\varepsilon\frac{1}{N}\sum_{i}^{N}\frac{df(x_i,w)}{dw}$

缺点：每一次迭代都要用到所有的样本，计算量太大

SGD stochastic gradient descent

参数：学习率 $\varepsilon$

计算：随机选择 batch_size个样本，带入梯度公式，求平均后更新参数

$w = w-\varepsilon\frac{1}{N}\sum_{i}^{batch\_size}\frac{df(x_i,w)}{dw}$

缺点: 学习率不变的情况下不好收敛,因为随机选取，跟真实梯度肯定存在误差。

学习率衰减：

$\varepsilon_{new}=\frac{\varepsilon_0}{1+decay*\varepsilon_{old}}$

其中 $decay$ 为衰减率，在很多框架中提供该参数，一般设置为0.9

Momentum

参数：学习率$\varepsilon$,动量衰减率$\alpha$,初始动量$v$

计算：

  1. 随机选择 batch_size个样本，带入梯度公式，求平均
  2. 更新速度$v$
  3. 跟新参数$w$
$v_{new} = \alpha*v_{old}-\varepsilon*dw$
$dw = dw + v_{new}$

特点: 应对SGD更新震荡的情况，前一轮梯度与当前轮梯度方向相同时，能加速更新，方向相反时，能抑制震荡。

Nesterov Momentum

参数：学习率$\varepsilon$,动量衰减率$\alpha$,初始动量$v$

计算：

  1. 随机选择 batch_size个样本
  2. 用新参数 $w+\alpha*v_{old}$ ，计算梯度，求平均
  3. 更新速度$v$
  4. 跟新参数$w$
$dw = getGrad(X_{batch},w+\alpha*v_{old})$
$v_{new} = \alpha*v_{old}-\varepsilon*dw$
$dw = dw + v_{new}$

大致思路是用估计后的参数计算误差，迭代

AdaGrad

参数：学习率$\varepsilon$,稳定参数$\delta$

计算：

  1. 随机选择 batch_size个样本，带入梯度公式，求平均
  2. 叠加梯度
  3. 求出跟新值
  4. 跟新参数
$r = r +||dw||^2$
$\Delta\theta = -\frac{\varepsilon}{\delta+\sqrt{r}}*dw$
$\theta = \theta+\Delta\theta$

观察可以发现，当当前轮的梯度$dw$大时，那么更新率$\frac{\varepsilon}{\delta+\sqrt{r}}$ 会小一点

缺点：深度学习中，深度过深会造成训练提前结束

RMSProp

参数：学习率$\varepsilon$,稳定参数$\delta$,衰减率$\rho$

中间变量$r$ 初始化为0

计算：

  1. 随机选择 batch_size个样本，带入梯度公式，求平均
  2. 叠加梯度
  3. 求出跟新值
  4. 跟新参数
$r = \rho r +(1-\rho)||dw||^2$
$\Delta\theta = -\frac{\varepsilon}{\delta+\sqrt{r}}*dw$
$\theta = \theta+\Delta\theta$

特点 : 通过引入新参数$\rho$ 解决训练提前结束的问题

Adam

Adam是最常用的优化算法，其本质上就是RMSProp加上一个动量后的算法

参数：学习率$\varepsilon$,稳定参数$\delta$,衰减率$\rho_1,\rho_2$

中间变量：s,r 分别对应一阶动量，二阶动量，初始化为0

计算：

  1. 叠加梯度，给一阶动量s,和二阶动量r
  2. 将s,r乘上系数后应用到RMSProp的更新式中
  3. 更新$w$
$s = \rho_1 s +(1-\rho_1)dw$
$r = \rho_2 r +(1-\rho_2)||dw||^2$
$s^* = \frac{s}{1-\rho_1}$
$r^* = \frac{r}{1-\rho_2}$
$\Delta w = -\frac{s^*}{\delta+\sqrt{r^*}}$
$w = w+\Delta w$

化简后可以看出：

$\Delta w = -\frac{\frac{\rho_1}{1-\rho_1}s+dw}{\delta+\sqrt{r}}\\=k*f(w_{old})+\xi*dw$

本质上就是动量加上梯度的形式
