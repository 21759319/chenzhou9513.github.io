mybatis逆向工程 配置

:dog: org.mybatis.generator 工具

逆向生成pojo和mapper

配置文件和生成方法

依赖

  1. 新建maven工程
  2. pom中加入依赖

配置

在resources下新建generatorConfig.xml

运行

Idea 下：

  1. run —> Edit Configuration —> +(add configuration) —> maven
  2. 选择Working directory 后, Command line: mybatis-generator:generate
  3. 运行generate
