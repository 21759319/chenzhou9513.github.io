前向传播

$\mathrm { E } [ x ] = \frac{1}{N}\sum{x_i}$
$Var[x] = \frac{1}{N}\sum{(x_i-E[x])^2}$
$\widehat { x } = \frac { x - \mathrm { E } [ x ] } { \sqrt { \operatorname { Var } [ x ] + \epsilon } }\space\space\space --(1)$
$x_{normal} = \gamma \widehat { x }+\beta \space\space\space--(2)$

其中(1) 是将样本输入的分布 转换成标准正太分布N(0,1)，分布加上$\epsilon$ 是为了防止分母过小或是0。

(2) 是因为引入BN后神经网络的表达能力可能会下降，实际上(2)式就是对(1)式的反向操作，引入两个可训练的参数$\gamma,\beta$，抵消这种影响，增强神经网络的表达能力。

前向传播时还要做一些额外的操作，我们在训练时，前向传播用的是batch_size个样本去做，那么可以很容易地计算mean和var,但是在测试集上，一次只传一个数据，计算mean和var就没有意义了,一种做法是将训练集的mean和var运用到测试集

具体做法是，在每一轮batch前向传播时：

$running_mean = \xirunning_mean+(1-\xi)batch_mean$

$running_var = \xirunning_var+(1-\xi)batch_var$

最后预测的测试集的时候，就用$running_mean,running_var$ 代替上述的$\mathrm { E } [ x ],Var[x] $计算

反向传播

$dout:[N,D]\\x:[N,D]\\\gamma:[D]\\\beta:[D]\\input:前一层的输出$

$d\beta = dout\frac{df()}={d\beta} \sum_{}{dout_i}$

$d\gamma = dout\frac{df(+)}{df()}\frac{df()}{d\gamma} = \sum_{N}(input_i*dout_i) \space\space\space对应维度相乘$

$dx : 根据反向传播累乘$
